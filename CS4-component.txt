COMPONENT BREAKDOWN
===================

A. SIDEBAR COMPONENTS
   ├── Toggle Button (Panel Splitter)
   ├── Top Navigation Section
   │   ├── New Chat Button
   │   └── Search Button (with shortcut)
   ├── Features Section
   │   ├── Narrative OS
   │   ├── Knowledge/Insights
   │   └── Folders
   ├── Chat History Section
   │   ├── Section Header (with count)
   │   ├── Chat List (scrollable)
   │   └── Chat Item Menu (ellipsis)
   └── User Account Section
       ├── Profile Display
       ├── Plan Indicator
       └── Dropdown Menu

B. MAIN CONTENT COMPONENTS
   ├── Mind Map Canvas
   │   ├── Node Components
   │   ├── Connection Lines
   │   ├── Zoom Controls
   │   └── Context Menu
   └── Task Management Area
       ├── Swimlane Containers
       │   ├── High Priority
       │   ├── Medium Priority
       │   └── Low Priority
       ├── Task Cards
       ├── Drag & Drop Handlers
       └── Filter Controls

C. MODAL COMPONENTS
   ├── Quick Capture Modal
   │   ├── Text Input Area
   │   ├── Media Input Buttons
   │   └── Submit Button
   ├── Search Modal
   │   ├── Search Input
   │   ├── Results List
   │   └── Keyboard Navigation
   ├── Settings Modal
   │   ├── General Settings
   │   ├── Notification Settings
   │   └── AI Preferences
   └── Daily Ritual Modals
       ├── Morning Briefing
       └── Evening Reflection

D. FLOATING COMPONENTS
   ├── Today's Focus Panel
   │   ├── Priority Sections
   │   ├── Task Summary
   │   └── Quick Actions
   ├── Notification System
   └── Context Menus

E. TECHNICAL COMPONENTS
   ├── Service Worker (Offline)
   ├── IndexedDB Manager
   ├── Sync Engine
   ├── AI Integration Layer
   ├── Voice Recognition
   ├── OCR Processing
   └── File Upload Handler

F. STATE MANAGEMENT
   ├── User Preferences
   ├── Chat History
   ├── Task Data
   ├── Mind Map Data
   ├── Offline Queue
